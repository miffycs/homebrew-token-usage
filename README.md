# homebrew-menubar-usage

Homebrew tap for [**menubar-usage**](https://github.com/miffycs/menubar-usage) — pin your Claude Code & Codex quota to the macOS menu bar.

## Install

```sh
brew tap miffycs/menubar-usage
brew install --cask menubar-usage
```

## Upgrade

```sh
brew upgrade --cask menubar-usage
```

## Uninstall

```sh
brew uninstall --cask menubar-usage
```
